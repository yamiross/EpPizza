# Pizza_Delivery
(Training Project)

Used technologies:
--Servlets
--JSP
--Spring Core
--Spring MVC
--Spring Security
--JPA(Hibernate)
--Jackson
--JUnit
--Mockito

User roles:
-Client
-Administrator

Client can:
-log in
-view & choose pizzas
-place order
-view his orders

Administrator can:
-log in
-view & edit pizzas
-add new pizzas
-view all orders
